Controlla le issues su GitHub

Ci sono tante cose da fare, assegnati una issue e mettiti a lavoro!

Aggiungi nuovi server

Acquista un server personalmente, oppure contatta università, provider ed istituzioni italiane e chiedi di mettere a disposizione un server.
Grazie a Docker puoi installare una nuova istanza in pochi minuti, guarda come fare.

Embedda il nostro codice

Copia il seguente snippet di codice per avere a disposizione un iframe da utilizzare ovunque tu voglia e per diffondere maggiormente la piattaforma

<div class="work_iframe">
    <iframe src="https://iorestoacasa.work/" height="70" width="120"
        style="border:0px;" name="iorestoacasa">
    </iframe>
</div>
<style>
    .work_iframe {
        position: relative;
        height: 0;
        overflow: hidden;
        max-width: 100%;
    }
    .work_iframe iframe {
        position: absolute;
        top: 0;
        left: 0;
        width: 100%;
        height: 100%;
    }
</style>
